Micro-interaction:

Link -

Ubisoft Store Website

Triggers -

User-initiated micro-interaction is triggered by hovering over the game card.

Rules -

hovering on:

  • The cursor changes to hand pointer when hovering over within the bounds of the game card

  • Game card slides up

  • The game card image has a dark opacity overlay(opacity about 75% or more) fades in

  • A 'heart with a circle'icon (top-right) & a 'windows'icon (bottom center) also fades in

hovering off:

  • The cursor changes from hand pointer to cursor when hovering off the game card

  • The game card slides back down

  • The game card image has a dark opacity overlay(opacity about 75% or more) fades out

  • A 'heart with a circle'icon (top-right) & a 'windows'icon (bottom center) also fades out
