Beta Description


Micro-interaction:

Link -

Ubisoft Store Website


Triggers -

Micro-interaction for Game card : is triggered by Hovering on & Hovering off the game card

Micro-interaction for Heart icon : is triggered by Hovering on & Hovering off the heart icon & clicking on the heart icon

Rules -

Game card - hovering on:

  • The cursor changes to hand pointer when hovering over within the bounds of the game card

  • Game card rotates (180 deg) to the back of the card, around the y-axis in a gradient cardholder(flips left to right)

  • Back of the game card is displayed with 'white outlined heart' icon (top - right), White 'windows' icon (middle - center), and card information: title & sub-title & price(bottom - left)

Game card - hovering off:

  • The cursor changes from hand pointer to cursor when hovering off the game card

  • Game card rotates back to the front image side (180 deg), around the y-axis in a gradient cardholder (flips right to left)

  • Front of the game card is displayed with the game card image

Heart Icon - hovering on:

  • The cursor changes to hand pointer when hovering over within the bounds of the heart icon

  • Heart icon is filled with blush pink color fading in

Heart Icon - hovering off:

  • The cursor changes from hand pointer to cursor when hovering off the heart icon

  • Heart icon filled with blush pink color returns to the transparent fill

Heart Icon - on click

  • The cursor changes from hand pointer to cursor when hovering on the heart icon

  • Heart icon filled with blush pink color fading in, as the blush pink enlarges & returns to its existing size with red color (pulse like movement).

Loops & Modes -

Hover on - Game card:

No Loops, a 'white outlined heart' icon, a 'white windows' icon, and card information (title, subtitle & price) appear as modes.

Hover on - Heart Icon:

No loops, but mode of new color of blush pink fades in while hovering over the heart icon.

Click on - Heart Icon:

No loops, but mode of new color of red fades in, as heart icon size increase and decreases after clicking on the heart icon.


Stages of Micro-interactions:

Stage 1 - hover off game card

Stage 2 - hover on game card

Stage 3 - hover on heart icon

Stage 4 - hover off heart icon

Stage 5 - click on heart icon

Stage 6 - hearting: heart icon enlarges

Stage 7 - hearted icon
