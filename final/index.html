Final Description


Micro-interaction:


Link -

Ubisoft Store Website


Triggers -

Game card : Hover on & Hover off

Heart icon : Hover on & Hover off

Heart icon : on 1st & 2nd click

Windows Icon : hover on & off

Price text Element: hover on & hover off

Rules -

Game card - hovering on:

  • Game card rotates (180 deg) to the back of the card, around the y-axis in a gradient cardholder(flips left to right)

  • Back of the game card is displayed with 'white outlined heart' icon (top - right), White 'windows' icon (middle - center), and card information: title & sub-title & price(bottom - left)

Game card - hovering off:

  • Game card rotates back to the front image side (180 deg), around the y-axis in a gradient cardholder (flips right to left)

  • Front of the game card is displayed with the game card image

Heart Icon - hovering on:

  • The cursor changes to hand pointer when hovering over within the bounds of the heart icon

  • Heart icon is filled with blush pink color fading in

Heart Icon - hovering off:

  • The cursor changes from hand pointer to cursor when hovering off the heart icon

  • Heart icon filled with blush pink color returns to the transparent fill

Heart Icon - on 1st click:

  • Heart icon is filled with blush pink color, then smoothly transitions fading to red (scales up by nearly 10%) & returns to its existing size with red color (pulse like movement).

  • Database is updated for 1st click for user's favorited game products.

Heart Icon - on 2nd click:

  • Heart icon filled with red color fades out, and returns to a 'white outlined heart' icon.

  • Database is updated for 2st click for user's favorited game products.

Windows Icon - Hover on:

  • The cursor changes to hand pointer when hovering over within the bounds of the window icon

  • Windows icon moves up and down during the animation, lasts for 3 seconds, repeats infinitely while hovering on.

  • Animation starts from the original position, and the element moves down by nearly 35 pixels at the approximately 65% mark, and then returns it to the original position.

Windows Icon - Hover off:

  • The cursor changes from hand pointer to cursor when hovering off the windows icon

  • Animation stops and the windows icon returns it to the original position in its original state.

Price text Element - Hover on:

  • The cursor changes to hand pointer when hovering over within the bounds of the price text element.

  • Price text element color changes to golden beige on hover (smoothly transitions with a duration of 0.5 seconds).

  • A text shadow(white) is seen, creating a glowing effect with multiple layers

  • The margin-right and margin-left properties are adjusted on hover, creating a shifting effect while hover on

Price text Element - Hover off:

  • The cursor changes from hand pointer to cursor when hovering off the price text element.

  • The price text element will retain its initial size, color, and margin properties (original state).

Loops & Modes -

Game card:

No Loops, a 'white outlined heart' icon, a 'white windows' icon, and card information (title, subtitle & price) appear as modes.


Heart Icon:

No loops & Modes


Windows Icon:

No modes, but floating animation loops while hovering on.


Price:

No loops & Modes
